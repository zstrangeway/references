# Kubectl Commands

[<- Home](https://github.com/zstrangeway/references)

## Contexts

- ### Get Contexts
    `kubectl config get-contexts`

- ### Switch Contexts
    `kubectl config use <context>`

## Nodes

- ### List Nodes
    `kubectl get nodes`

## Deployments

- ### Create Deployment
    `kubectl create deployment <name> --image=<image name>`

- ### List Deployments
    `kubectl get deployment`

## Pods

- ### List Pods
    `kubectl get pod`
